Detailed Description

An abstraction for editors working on Salt SDocumentGraphs.

This class is intended to be extended by clients implementing new editors that work on SDocumentGraphs. The input for editors extending this class must be a SaltXML file containing the persisted document graph. The input type must be FileEditorInput. The editor retains a single instance instance of SDocumentGraph which contains the contents of the input file.
