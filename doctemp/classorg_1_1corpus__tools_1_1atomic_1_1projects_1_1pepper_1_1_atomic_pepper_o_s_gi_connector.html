Detailed Description

This class is an implementation of Pepper.

It bridges between the Atomic environment (Eclipse RCP based on an Equinox OSGi environment), and the Pepper OSGi environment, which is forced to use Atomic's OSGi BundleContext. The AtomicPepperOSGiConnector is used to install and start bundles as well as update Pepper module bundles.
