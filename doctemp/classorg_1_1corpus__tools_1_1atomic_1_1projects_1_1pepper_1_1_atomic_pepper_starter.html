Detailed Description

This class configures and starts an instance of Pepper (implemented in AtomicPepperOSGiConnector.

It also stores the Pepper instance in AtomicPepperStarter#pepper, which can be got from the instantiators of this class via AtomicPepperStarter#getPepper() to perform actions on the Pepper object.
