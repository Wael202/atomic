Detailed Description

This class represents all properties that Pepper as used in Atomic has.

The properties themselves are loaded from the "pepper.properties" file. The location of the file is FIXME in the configuration folder of the Pepper home directory as defined by findPepperHome().

The properties also include information about the location of paths needed for bundle-level operations.
